## 게임 조작방법
* 장애물(풍선, 게, 팝콘, 나비)을 피해서 무사히 도착지까지 가세요!
* 좌우 움직이기 ← →
* 점프 : ↑
* 멈추기 : ↓
* 장애물 없애기 : Space + ← / → / ↑ (스페이스바와 함께 눌러요)
